Política de Privacidade

BTG Pactual

1. Introdução

A BTG Pactual está comprometida em proteger a privacidade e segurança dos dados de nossos usuários. Esta Política de Privacidade descreve como coletamos, usamos, armazenamos e protegemos suas informações ao utilizar nosso aplicativo de ponto de venda (POS).

2. Dados Coletados

Coletamos as seguintes categorias de dados:

  • Dados de Transação: Valor, data, hora, método de pagamento
  • Dados do Dispositivo: Modelo do POS, versão do sistema
  • Dados de Uso: Funcionalidades utilizadas, frequência de uso
  • Dados de Autenticação: Email e senha criptografados

3. Como Usamos seus Dados

Utilizamos seus dados para:

  • Processar transações de pagamento
  • Melhorar nossos serviços e funcionalidades
  • Fornecer suporte técnico
  • Cumprir obrigações legais e regulatórias
  • Prevenir fraudes e atividades suspeitas

4. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros, exceto:

  • Com processadores de pagamento para completar transações
  • Quando exigido por lei ou ordem judicial
  • Com seu consentimento explícito

5. Segurança dos Dados

Implementamos medidas de segurança técnicas e organizacionais:

  • Criptografia de ponta a ponta para dados sensíveis
  • Autenticação multifator
  • Monitoramento contínuo de segurança
  • Conformidade com padrões PCI-DSS

6. Retenção de Dados

Mantemos seus dados apenas pelo tempo necessário:

  • Dados de transação: 5 anos (requisito legal)
  • Logs de sistema: 90 dias
  • Dados de conta: Até solicitação de exclusão

7. Seus Direitos

Você tem direito a:

  • Acessar seus dados pessoais
  • Corrigir dados incorretos
  • Solicitar exclusão de dados
  • Exportar seus dados
  • Revogar consentimentos
